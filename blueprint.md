# 로또 번호 추천 웹사이트

## 개요

이 애플리케이션은 사용자에게 한 번에 5세트의 로또 번호를 추천하는 웹사이트입니다. 각 세트는 1부터 45까지의 숫자 중 6개의 고유한 숫자로 구성됩니다.

## 디자인 및 기능

- **UI:** 현대적이고 미니멀한 디자인을 적용하여 사용자 경험을 향상시킵니다.
- **색상 팔레트:** 어두운 배경과 대비되는 생생한 색상을 사용하여 번호를 강조합니다.
- **타이포그래피:** 가독성 높은 글꼴을 사용하여 정보를 명확하게 전달합니다.
- **인터랙션:** 사용자가 '번호 생성' 버튼을 클릭하면 5개의 새로운 로또 번호 세트가 부드럽게 표시됩니다.
- **반응형 디자인:** 웹사이트는 데스크톱과 모바일 장치 모두에서 잘 보이도록 반응형으로 제작됩니다.

## 구현 계획

1.  **HTML (`index.html`):**
    *   웹사이트의 기본 구조를 설정합니다.
    *   제목, 번호 생성을 위한 버튼, 생성된 번호가 표시될 컨테이너를 포함합니다.
    *   `style.css`와 `main.js` 파일을 연결합니다.

2.  **CSS (`style.css`):**
    *   전체적인 레이아웃, 색상, 글꼴 등 웹사이트의 스타일을 지정합니다.
    *   버튼과 번호 표시에 애니메이션 효과를 추가하여 동적인 느낌을 줍니다.

3.  **JavaScript (`main.js`):**
    *   1부터 45까지의 숫자 중에서 6개의 고유한 무작위 번호를 생성하는 함수를 구현합니다.
    *   버튼 클릭 시 5개의 번호 세트를 생성하고 화면에 표시하는 이벤트 리스너를 추가합니다.
